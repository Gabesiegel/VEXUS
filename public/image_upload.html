CSV Batch Upload Instructions

Upload multiple images using a CSV file with the following headers:

filePath,title,veinType,waveformType,quality,tagVeinType,tagWaveform,description,contributor,institution,email

Required fields: filePath, title, veinType, waveformType

Optional fields: quality, tagVeinType, tagWaveform, description, contributor, institution, email

Tag Fields Format:

The tag fields are split into three separate columns:

  • quality: High, Medium, or Low
  • tagVeinType: Hepatic, Portal, Renal, or IVC
  • tagWaveform: Normal, Mild, or Abnormal

    • Example row: "/path/to/images/hepatic_example.png,Normal Hepatic Vein,Hepatic,Normal,High,Hepatic,Normal,This is a normal hepatic vein showing triphasic flow pattern,Dr. John Smith,University Hospital,john.smith@example.com"

    Note: Tag fields are optional. You can leave them blank if not applicable.

The filePath should point to local images on your computer. All images referenced will be uploaded.
